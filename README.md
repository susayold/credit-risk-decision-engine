Interactive website with English and Vietnamese views. The live project is the recommended starting point.

This project develops an explainable credit scorecard and decision framework from target definition through WOE/IV, probability of default, scorecard points, out-of-time validation, cutoff economics and model governance.

## Key Results

- 1.348 million core accepted/booked accounts assessed.
- 331,865 enriched accepted loans used for scorecard development.
- 0.781 out-of-time test AUC for the clean WOE-logistic scorecard.
- 0.418 out-of-time test KS for the clean scorecard.
- 100,000-row validation sample included for review.

## What The Project Demonstrates

- Matured good/bad target design and leakage control.
- Transparent binning, Weight of Evidence and Information Value analysis.
- Traditional WOE-logistic scorecard construction and score scaling.
- AUC, KS, calibration, Brier score and stability validation.
- Approval, bad-rate, bad-capture and expected-loss cutoff trade-offs.
- Model card, limitations, fairness/proxy review and governance controls.

## Analytical Boundary

The scorecard is a portfolio case study built from public accepted/booked-account data. Rejected applications have no observed repayment outcomes and are used only for selection-bias sensitivity and funnel context. No production lending decision or bank-approved model is claimed.
